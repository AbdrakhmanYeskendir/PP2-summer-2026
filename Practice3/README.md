# Practice 3

## Description

This practice contains Python examples for functions, lambda expressions, classes, and inheritance.

## Topics Covered

### Functions

- Basic functions
- Function arguments
- Return values
- `*args` and `**kwargs`

### Lambda

- Lambda basics
- Lambda with `map()`
- Lambda with `filter()`
- Lambda with `sorted()`

### Classes

- Class definition
- `__init__()` method
- Instance methods, class methods, and static methods
- Class variables and instance variables

### Inheritance

- Basic inheritance
- `super()` function
- Method overriding
- Multiple inheritance
